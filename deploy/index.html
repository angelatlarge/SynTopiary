SynTopiary

To make a syntactic tree:
  1. Type a tree formula
    • parenthesis begin and children
    • commas separate children
    • use a backslash to make textual commas or parenthesis
  2. Hit enter
  3. If it doesn't look good, go back to step 1
  4. Click "Generate", select output file type, and save the file to your computer
  5. Insert the file into your favorite word processor
